望年会 招待状 — ファイル構成と使い方
====================================

【フォルダ構成】
  index.html          ... 招待状本体（このファイルを開けば動きます）
  assets/
    hero.jpg          ... レッドカーペット写真（こすって出てくる画像）
    mark.png          ... 会社ロゴマーク
    wordmark.png      ... 社名ロゴ（今日を生きる 日本は生きる / Today Japan）

【使い方】
  1. このフォルダごとサーバーにアップロードしてください。
     （index.html と assets フォルダは必ず同じ階層に置く）
  2. index.html のURLを共有すれば、そのまま招待状として使えます。

【よく使う編集ポイント】index.html を開いて書き換えるだけ
  ・合言葉を変える:
      <script> 内の  const PASSWORD = "おむすび";  の "おむすび" を変更
  ・日時・会場・参加費などの文言:
      該当する日本語テキストを検索して書き換え
  ・写真を差し替える:
      assets/hero.jpg を同じファイル名で置き換え

【注意】
  ・フォントはGoogle Fontsを読み込んでいます（インターネット接続時に自動表示）。
  ・index.html と assets/ の位置関係（相対パス）は変えないでください。
